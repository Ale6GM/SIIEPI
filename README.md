## SIIEPI

SIIEPI: Es una Aplicacion Web que tiene como objetivo gestionar el proceso de informatización en una empresa brindando esticto control sobre los medios de computo distribuidos por areas de trabajo.
En el commit Inicial gestionará:

- Computadoras
- UPS



## Gestor de Bases de Datos

SIIEPI usa como motor de bases de datos MYSQL
